AI Data Analysis Assistant 🤖
A simple AI-powered Data Analysis Assistant built with Python, Streamlit, and OpenRouter API.

Features
✅ Load and preview CSV datasets
✅ Automatic statistical analysis (Mean, Max, Min, Counts)
✅ Data Visualization (Bar, Histogram, Pie, Scatter) with proper Titles and Labels
✅ AI-powered chart explanations
✅ Ask natural language questions about the data

Setup Instructions
Install requirements: pip install -r requirements.txt
Add your API key to a .env file: OPENROUTER_API_KEY=your_key_here
Run the app: streamlit run main.py
Project Structure
main.py - Core application logic, UI, and AI integration
dataset.csv - Sample IMDB movies dataset
requirements.txt - Python dependencies
README.md - Project documentation
